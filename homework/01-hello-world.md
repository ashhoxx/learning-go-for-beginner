# Домашнее задание 1: «Привет, мир!»

## Цель
Научиться создавать и запускать простейшую программу на Go.

## Шаги
1. Установи Go с официального сайта: https://go.dev/dl/
2. Создай файл под названием `hello.go`
3. Вставь в него следующий код **точно как написано**:

```go
package main

import "fmt"

func main() {
    fmt.Println("Привет, мир!")
}
```

4. Открой терминал (или PowerShell) в той же папке, где лежит `hello.go`
5. Выполни команду:
```bash
go run hello.go
```
6. Убедись, что в терминале появилась строка:
```
Привет, мир!
```

## Что прислать
Сделай **скриншот терминала**, на котором видно:
- команду `go run hello.go`
- строку `Привет, мир!` в ответ

Пришли этот скриншот **в ответ на пост с уроком** в Telegram-канале:  
→ https://t.me/go_practical

## Советы
- Если возникает ошибка — внимательно сравни свой код с примером.
- Проверь:
  - `package main` (строчные буквы)
  - `import "fmt"` (в двойных кавычках)
  - `func main()` (без опечаток)

> Не присылай код — присылай **скриншот терминала**.  
> Это доказывает, что ты запустил программу, а не скопировал текст.


После этого всё будет отображаться **ровно, как у официальных проектов**.
