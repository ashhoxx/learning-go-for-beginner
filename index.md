# Практический Go — уроки для новичков

Этот курс для тех, кто хочет **освоить Go с нуля через реальный код**, а не через теорию.

## Как учиться
1. Прочитай урок
2. Выполни задание локально
3. Пришли результат в Telegram: [@go_practical](https://t.me/go_practical)

> Без практики — нет прогресса. Точка.
